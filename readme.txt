== Description ==

**Informizely – Complete survey platform for customer feedback**

Get to the “why” of your customers by surveying them in your website, web apps and emails. Informizely helps you to quickly get winning insights into your customers motives. To improve your website, product, service and conversion rate.

Install the Informizely code once on your WordPress site to show beautiful targeted surveys and/or feedback buttons on specific pages.
See [https://www.informizely.com/product](https://www.informizely.com/product) for an overview of all features.

== Frequently Asked Questions ==

= Do I need an Informizely account to use this plugin? =

Yes you do. You can start a free trial here: [https://app.informizely.com/register](https://app.informizely.com/register).

This plugin loads the Informizely configuration file for your site, so that the surveys that you configure in your Informizely dashboard are shown on your WordPress site.

For more frequently asked questions see this page: [https://www.informizely.com/help/faq](https://www.informizely.com/help/faq).

== Installation ==

1. Upload `informizely` to the `/wp-content/plugins/` directory, or add it from the plugin menu.
2. Activate your plugin.
3. Go to Settings - Informizely.
4. Enter the Site ID of your site as shown in your [Informizely dashboard](https://app.informizely.com/Home/SiteIds).
